## JavaScript

- 0은 fasly value
- console.table (테이블 형태로 출력) console.dir (객체 속성을 계층구조로 출력)
- input의 value는 항상 string 형태로 출력됨
- valueAsNumber는 number 형태로 출력, valueAsDate는 date 형태로 출력
- async(background 로딩, 로드되면 바로 실행), defer(background 로딩, html 읽은 후 실행)
